# 近くの図書館を探す

カーリル図書館APIを使って、現在地または住所から近くの図書館を探し、蔵書の貸出状況を確認できるWebアプリです。

## 概要

- 現在地（GPS）または住所・地名から近くの図書館を検索
- 図書館を選択してISBNで蔵書検索
- 貸出状況（貸出可・貸出中など）と予約ページへのリンクを表示

## 使い方

1. 「現在地から探す」または「住所・地名から探す」タブを選択
2. 図書館一覧からチェックボックスで図書館を選択
3. ISBNを入力して「蔵書検索」ボタンをクリック

> 住所検索は「東京都渋谷区」のように都道府県名から始めると精度が上がります

## 技術スタック

- HTML / CSS / JavaScript（バニラJS）
- [カーリル図書館API](https://calil.jp/doc/api_ref.html)
